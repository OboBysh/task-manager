\# Task Manager CLI



Простое консольное приложение для управления списком дел.  

Разработано в учебных целях с разделением на модули, тестами и документированием.



\*\*Репозиторий:\*\* \ https://github.com/OboBysh/task-manager



\---



\## 📦 Установка



1\. Клонируйте репозиторий:

&#x20;  ```bash

&#x20;  git clone https://github.com/OboBysh/task-manager.git

&#x20;  cd task-manager

2\. Установите зависимости (рекомендуется использовать виртуальное окружение):

&#x09;pip install -r requirements.txt



Запуск

Приложение запускается через точку входа main.py с командами:



Добавить задачу:

&#x09;python main.py add "Купить молоко" -p высокий --due 2024-12-31

Параметры:

description – описание задачи (обязательно)

\-p, --priority – приоритет: низкий, средний (по умолчанию), высокий

\--due – срок выполнения в формате ГГГГ-ММ-ДД (опционально)



Просмотреть все задачи:

&#x09;python main.py view

Отметить задачу выполненной:

&#x09;python main.py complete <ID>

Пример: python main.py complete 1



Удалить задачу:

&#x09;python main.py delete <ID>

Вызов справки:

&#x09;python main.py -h



&#x20;Структура проекта

task-manager/

├── taskcli/                # основной пакет

│   ├── \_\_init\_\_.py

│   ├── storage.py          # работа с JSON-файлом задач

│   ├── logic.py            # бизнес-логика (добавление, удаление, завершение)

├── tests/                  # автоматические тесты (pytest)

│   ├── \_\_init\_\_.py

│   ├── test\_storage.py

│   ├── test\_logic.py

│   └── test\_cli.py

├── main.py                 # точка входа в приложение

├── README.md               # документация

├── requirements.txt        # зависимости

└── .gitignore



Тестирование

Для запуска тестов выполните:

&#x09;pytest tests/

Тесты покрывают работу с хранилищем, бизнес-логику и базовый вызов CLI.
